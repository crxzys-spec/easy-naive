# EasyNaive 0.1.1

## Release Title

EasyNaive 0.1.1

## Highlights

- Modernized WinForms UI with custom header, refined node list, compact controls, status icons, and cleaner log viewer.
- Added split latency columns: `Ping` for direct TCP connect to node entry and `URL Test` for real proxy-path testing through sing-box.
- Added LAN proxy opt-in: `Allow LAN connections` can expose the mixed HTTP/SOCKS proxy on `0.0.0.0`.
- Improved node sharing/import workflow, subscription handling, right-click node actions, and manual/auto hot switching.
- Improved TUN elevation flow with service support to reduce repeated UAC prompts.
- Added rotating logs and clearer diagnostics/error messages.
- Updated sing-box DNS config generation for current sing-box versions.

## Requirements

- Windows x64.
- Microsoft .NET Desktop Runtime 8 x64.
- Windows Firewall inbound rule may be required when `Allow LAN connections` is enabled.

## Known Limits

- TUN mode depends on Windows networking and elevated service availability.
- LAN proxy access is disabled by default and must be explicitly enabled in Settings.
- `URL Test` requires sing-box core to be running; `Ping` can be tested without an active connection.
