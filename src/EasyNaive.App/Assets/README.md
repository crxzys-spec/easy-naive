# EasyNaive App Assets

Put the application icon here:

```text
App.ico
TrayConnected.ico
TrayError.ico
TrayStopped.ico
```

Recommended `.ico` sizes:

```text
16x16
24x24
32x32
48x48
64x64
256x256
```

`App.ico` is used for the Windows executable icon, main window icon, and default tray icon.
`TrayConnected.ico` is used for the tray icon when the core is connected.
`TrayError.ico` is used for the tray icon when the core is in an error state.
`TrayStopped.ico` is used for the tray icon when the core is disconnected.
