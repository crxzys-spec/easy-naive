# Code of Conduct

EasyNaive is not yet open for broad public contribution. Until then, use the
following minimal conduct rules for issues, discussions, and contributions.

## Expected Behavior

- Be respectful and specific.
- Keep technical discussions focused on reproducible facts.
- Assume maintainers and contributors are acting in good faith.
- Avoid sharing private credentials, subscription links, or personal data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing another user's private data or credentials.
- Spam or unrelated promotion.
- Intentionally submitting malicious code, configs, or binaries.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate
these rules. Severe violations may result in blocked participation.

Before a public open-source release, replace this file with a complete code of
conduct if the project expects community participation at scale.
